# SimpleCore

Make some chairs in Minecraft.

## Screenshot

![0.png](screenshot/0.png)

## Usage

Right click on any kind of **stairs** with **empty** main hand.

## Permission

- **nekochairs.use**: Allow players sit on chairs.

## Build

```bash
git clone https://github.com/neko-craft/NekoCharis.git

gradlew jar
```
